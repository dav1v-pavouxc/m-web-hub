# m-web-hub

## 项目简介

m-web-hub 是一个用于归档和发布独立 HTML 页面的仓库。这里存放各类静态网页资源，方便统一管理和访问。项目本身不针对任何特定域名或网站，仅作为页面集合的载体。

## 目录说明

```
m-web-hub/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的公共资源（样式、脚本、图片等）
├── index.html      # 项目入口页（可选）
└── README.md       # 本文件
```

- `pages/`：每个子文件夹或文件对应一个独立页面，命名清晰以区分不同功能。
- `assets/`：包含页面间可能共享的 CSS、JavaScript 或媒体文件，避免重复存放。

## 页面归档说明

本仓库收录的 HTML 页面均为独立打包的静态文件，无需后端支持即可直接打开。页面内容遵循以下原则：

- 所有页面均为自包含或引用公共资源，不依赖外部服务。
- 归档的页面可能涉及工具、演示、实验性项目或信息展示等用途。
- 页面之间相互独立，无关联性要求。

## 维护说明

- **添加页面**：在 `pages/` 下新建文件夹，放入 HTML 及相关资源，如有公共依赖可放入 `assets/`。
- **更新页面**：直接修改对应文件，保持版本记录可追溯。
- **移除页面**：删除对应文件夹及入口链接。
- 欢迎提交 Issue 或 Pull Request 来改进页面或调整结构。

## 使用方式

1. 克隆本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件，或通过项目根目录的 `index.html` 导航访问。
3. 也可部署到任意静态托管平台（如 GitHub Pages），通过在线链接浏览。
